这里是各个参数的解释：

  1. base： 待排序数组的起始地址（指向数组的指针）。
  2. nmemb： 数组中元素的数量。
  3. size： 每个元素的大小（以字节为单位）。
  4. compar： 比较函数的指针，用于定义元素的顺序。比较函数的原型应该是 int compar(const void *a, const void *b)，并且满足以下条件：
    • 如果 a 应该排在 b 前面，返回负值。
    • 如果 a 和 b 相等，返回零。
    • 如果 a 应该排在 b 后面，返回正值。
